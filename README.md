# FaceKeeper — Система контроля доступа на основе распознавания лиц

## 📌 О проекте

**FaceKeeper** — это корпоративная система контроля доступа, использующая биометрическую идентификацию по лицу. Система позволяет регистрировать сотрудников, управлять уровнями доступа, отслеживать вход/выход и администрировать рабочие помещения с разными требованиями безопасности.

### 🔍 Основной сценарий использования

1. Сотрудник подходит к камере
2. Система захватывает изображение, детектирует лицо (YOLOv8) и извлекает эмбеддинг (InsightFace)
3. Поиск в векторной базе FAISS для идентификации
4. Проверка уровня доступа к конкретной зоне
5. Логирование события с геолокацией (IPinfo)
6. Предоставление или запрет доступа

---

## 🚀 Ключевые возможности

| Функция | Описание |
|---------|----------|
| **Распознавание лиц** | InsightFace (buffalo_l) для высокоточных эмбеддингов (512d) |
| **Детекция лиц** | YOLOv8 с кастомной моделью для выделения одного лица |
| **Векторный поиск** | FAISS IndexFlatIP для быстрого поиска по косинусному сходству |
| **Хранение данных** | MongoDB (сотрудники, рабочие места, логи) |
| **Сжатие изображений** | zlib + Binary BSON для эффективного хранения |
| **Geo-логирование** | IPinfo для определения местоположения входа |
| **Ролевая модель** | 6 уровней доступа (0–5) с разным набором команд |
| **Управление помещениями** | Рабочие места с capacity, required_access_level |
